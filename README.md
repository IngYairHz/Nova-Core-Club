# 🚀 Nova Core Club

## 👋 Sobre nosotros

Nova Core Club es un club estudiantil enfocado en el desarrollo de software, la tecnología y la innovación.

Nuestro propósito es crear un espacio donde estudiantes puedan aprender, colaborar y desarrollar proyectos tecnológicos, fortaleciendo sus habilidades en programación y áreas relacionadas con la tecnología.

---

## 🎯 Objetivos

- Fomentar el aprendizaje de desarrollo de software
- Promover la colaboración entre estudiantes
- Crear proyectos tecnológicos
- Compartir conocimientos y recursos
- Impulsar el crecimiento académico y profesional de los integrantes

---

## 🛠️ Actividades

Dentro del club realizamos actividades como:

- Talleres de programación
- Proyectos colaborativos
- Retos de desarrollo
- Sesiones de aprendizaje
- Conferencias sobre tecnología
- Exploración de nuevas herramientas y tecnologías

---

## 💻 Áreas de interés

- Desarrollo de software
- Desarrollo web
- Programación
- Git y GitHub
- Inteligencia Artificial
- Bases de datos
- Tecnologías emergentes
